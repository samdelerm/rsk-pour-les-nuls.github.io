## Publication GitHub Pages

1. Crée un dépôt GitHub (ex : `rsk-pour-les-nuls`)
2. Pousse ce dossier à la racine du dépôt
3. Dans les paramètres du dépôt → Pages → Source : `main` branch, `/ (root)`
4. Le site est accessible à `https://ton-user.github.io/rsk-pour-les-nuls/`
