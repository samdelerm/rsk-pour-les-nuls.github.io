Python : Les bases

Variables, conditions, boucles, fonctions, maths — tout ce dont tu as besoin pour comprendre et écrire tes premiers programmes RSK.

1. Variables & types de données

Une variable, c'est une boîte dans laquelle tu ranges une valeur. En Python, tu n'as pas besoin de déclarer le type : Python le détecte tout seul.

variables.py
# Un entier (int)
score = 3

# Un nombre décimal (float)
position_x = 0.45       # en mètres sur le terrain
position_y = -0.25

# Une chaîne de caractères (str)
nom_equipe = "vert"

# Un booléen (bool)
balle_visible = True
robot_arrive = False

# Affichage
print(score)            # 3
print(position_x)       # 0.45
print(nom_equipe)       # vert
print(balle_visible)    # True

Dans RSK, les positions des robots et de la balle sont des float (nombres décimaux) exprimés en mètres. Par exemple, 0.45 signifie 45 centimètres.

Les types à connaître

TypeExempleUtilisation RSK
int3, -1, 255Numéro de robot, score
float0.45, -0.92Positions x/y, angles en radians
boolTrue, FalseRobot arrivé à destination ?
str"vert", "blue"Couleur d'équipe
NoneNoneBalle non détectée
tuple(0.5, 0.2, 3.14)Position (x, y, angle)

En Python, = sert à affecter une valeur, et == sert à comparer. Ne les confonds pas !

2. Opérations mathématiques

Python permet de faire des calculs directement. Pour les fonctions avancées (racine carrée, trigonométrie, etc.) on importe le module math.

maths.py
import math

# Opérations de base
a = 10 + 3     # addition       → 13
b = 10 - 3     # soustraction   → 7
c = 10 * 3     # multiplication → 30
d = 10 / 3     # division       → 3.333...
e = 10 // 3    # division entière → 3
f = 10 % 3     # modulo (reste)  → 1
g = 2 ** 8     # puissance       → 256

# Module math
print(math.sqrt(9))         # racine carrée → 3.0
print(math.pi)              # π ≈ 3.14159
print(math.radians(180))    # 180° en radians → 3.14159
print(math.degrees(math.pi))# π radians en degrés → 180.0
print(math.atan(1))         # arc tangente → 0.785 rad (45°)

# Valeur absolue (sans math)
print(abs(-5))              # → 5

Pourquoi les radians ? RSK utilise des radians pour les angles. 0 = regarde vers la droite, math.pi (≈3.14) = regarde vers la gauche. Pour convertir des degrés en radians : from math import radians puis radians(180).

Distance entre deux points

Une formule que tu utiliseras souvent pour calculer la distance entre un robot et la balle :

distance.py
import math

# Position du robot
rx, ry = 0.5, 0.2

# Position de la balle
bx, by = -0.3, 0.1

# Distance euclidienne
distance = math.sqrt((bx - rx)**2 + (by - ry)**2)
print(f"Distance robot→balle : {distance:.2f} m")
# → Distance robot→balle : 0.81 m

# Avec math.hypot (plus court)
distance = math.hypot(bx - rx, by - ry)

max() et min() — borner une valeur

max(a, b) renvoie le plus grand des deux, min(a, b) le plus petit. Ces fonctions sont intégrées à Python (pas besoin d'importer quoi que ce soit) et sont très utiles pour limiter une coordonnée à un intervalle.

max_min.py
# max : renvoie le plus grand
print(max(3, 7))       # → 7
print(max(-1, 0))      # → 0

# min : renvoie le plus petit
print(min(3, 7))       # → 3
print(min(-1, 0))      # → -1

# Cas classique RSK : limiter le Y d'un défenseur entre -0.30 et +0.30
by = 0.45   # la balle est loin sur le côté

by_clamp = max(-0.30, min(0.30, by))
# min(0.30, 0.45) → 0.30  (on coupe au plafond)
# max(-0.30, 0.30) → 0.30  (déjà dans la plage)
print(by_clamp)        # → 0.3

# Avec une valeur négative
by = -0.80
by_clamp = max(-0.30, min(0.30, by))
# min(0.30, -0.80) → -0.80
# max(-0.30, -0.80) → -0.30  (on coupe au plancher)
print(by_clamp)        # → -0.3
💡 Le « clamp » — patron universel

max(mini, min(maxi, valeur)) est la façon standard de bloquer une valeur dans un intervalle. On l'appelle clamp en anglais. Dans RSK tu verras souvent ce motif pour empêcher un robot de sortir du terrain ou de dépasser la zone de but.

3. Conditions : if / elif / else

Les conditions permettent à ton programme de décider quoi faire selon la situation. C'est le cœur de la logique de tes robots !

conditions.py
y_balle = 0.40   # position Y de la balle

# Structure if / elif / else
if y_balle > 0.30:
    print("La balle est au-dessus de la zone")
elif y_balle < -0.30:
    print("La balle est en-dessous de la zone")
else:
    print("La balle est dans la zone centrale")

# Opérateurs de comparaison
# >   plus grand que
# <   plus petit que
# >=  plus grand ou égal
# <=  plus petit ou égal
# ==  égal
# !=  différent

# Opérateurs logiques
x = 0.70
y = 0.10
if x > 0.60 and x < 0.92:
    print("Le robot est devant son but !")

if y > 0.30 or y < -0.30:
    print("La balle est hors de la zone centrale")

# Vérifier si la balle existe
balle = None   # ou (0.5, 0.2) si détectée

if balle is not None:
    print(f"Balle à ({balle[0]}, {balle[1]})")
else:
    print("Balle non détectée")

Dans RSK, client.ball vaut None quand la caméra ne voit pas la balle. Toujours vérifier if client.ball is not None: avant d'utiliser client.ball[0] !

Les conditions imbriquées

Tu peux mettre des if dans des if — c'est ce qu'on voit dans le code du défenseur RSK :

conditions_imbriquees.py
xball = 0.75
yball = 0.10

if yball > -0.30 and yball < 0.30:
    # La balle est dans la zone centrale (Y)
    if xball > 0.60 and xball < 0.92:
        # ET la balle est près du but
        print("Danger ! Le défenseur doit tirer !")
    else:
        print("Balle au centre - revenir en position")
else:
    # Balle en dehors : aller au coin le plus proche
    if yball >= 0.30:
        print("Aller à (0.80, 0.30)")
    else:
        print("Aller à (0.80, -0.30)")

4. Boucles : while & for

La boucle while — essentielle pour les robots

Tous les programmes RSK utilisent while True: pour tourner en permanence. Le robot répète ses commandes en boucle, environ des dizaines de fois par seconde.

while.py
# while True : boucle infinie (le robot travaille sans s'arrêter)
# ⚡ C'est la structure de BASE de tout programme RSK !

import rsk
from math import radians

with rsk.Client(host='localhost', key='') as client:
    while True:          # ← répète INDÉFINIMENT
        if client.ball is not None:
            bx = client.ball[0]
            by = client.ball[1]
            # Le robot suit la balle en permanence
            client.green1.goto((bx + 0.08, by, radians(180)), wait=False)

Le paramètre wait=False est crucial dans la boucle while True. Sans lui, le programme se bloquerait jusqu'à ce que le robot arrive à destination, et ne pourrait plus réagir si la balle bouge !

while avec condition

while_condition.py
compteur = 0
while compteur < 5:
    print(f"Tour numéro {compteur}")
    compteur += 1    # compteur = compteur + 1
# Affiche : Tour numéro 0, 1, 2, 3, 4

# break : sortir de la boucle immédiatement
x = 0
while True:
    x += 1
    if x >= 10:
        break    # on sort de la boucle
print(f"x vaut {x}")  # x vaut 10

# continue : passer à l'itération suivante
for i in range(10):
    if i % 2 == 0:
        continue    # on saute les nombres pairs
    print(i)        # affiche : 1, 3, 5, 7, 9

La boucle for

Moins utilisée dans RSK mais utile pour parcourir des listes :

for.py
# range(n) génère les entiers de 0 à n-1
for i in range(5):
    print(i)    # 0, 1, 2, 3, 4

# Parcourir une liste de positions
waypoints = [(0.3, 0.0), (0.0, 0.3), (-0.3, 0.0)]
for (x, y) in waypoints:
    print(f"Point ({x}, {y})")

5. Fonctions : def

Une fonction est un bloc de code réutilisable. Tu lui donnes un nom, des paramètres (les données en entrée) et elle peut retourner un résultat.

fonctions.py
import math

# ── Définition d'une fonction ──────────────────────────────
def distance(x1, y1, x2, y2):
    """Calcule la distance entre deux points."""
    return math.sqrt((x2 - x1)**2 + (y2 - y1)**2)

# ── Appel ──────────────────────────────────────────────────
d = distance(0.0, 0.0, 0.3, 0.4)
print(d)    # → 0.5

# ── Paramètres par défaut ──────────────────────────────────
def goto(x, y, angle=0):
    """Va à (x, y) avec l'angle donné (0 par défaut)."""
    print(f"Aller à ({x}, {y}) avec angle {angle}")

goto(0.5, 0.2)          # angle = 0 (par défaut)
goto(0.5, 0.2, 3.14)    # angle = π

# ── Fonction sans return ───────────────────────────────────
def afficher_balle(x, y):
    if x is None:
        print("Balle non visible")
    else:
        print(f"Balle en ({x:.2f}, {y:.2f})")

afficher_balle(0.45, -0.20)   # Balle en (0.45, -0.20)

Les fonctions dans RSK

Dans le code des équipes, on organise le comportement en deux fonctions : une pour le défenseur, une pour l'attaquant.

structure_rsk.py
import rsk
from math import radians

def Defenseur(xball, yball):
    """Le robot 2 (vert) défend son but."""
    if yball < 0.30 and yball > -0.30:
        # La balle est dans la zone → se positionner
        client.green2.goto((0.80, yball, radians(180)), wait=False)
    elif yball >= 0.30:
        client.green2.goto((0.80, 0.30, radians(180)))
    else:
        client.green2.goto((0.80, -0.30, radians(180)))

def Attaquant(xball, yball):
    """Le robot 1 (vert) attaque."""
    client.green1.goto((xball + 0.05, yball, radians(180)), wait=False)
    client.green1.kick()

# Boucle principale
with rsk.Client(host='localhost', key='') as client:
    while True:
        if client.ball is not None:
            xball = client.ball[0]
            yball = client.ball[1]
            Defenseur(xball, yball)
            Attaquant(xball, yball)

Attention à l'indentation (les espaces en début de ligne) ! En Python, c'est obligatoire. 4 espaces (ou 1 tabulation) pour chaque niveau. Une erreur de niveau = ton programme ne fonctionne plus.

6. Tuples & listes

Le tuple — le format de position RSK

Un tuple est une séquence immuable de valeurs entre parenthèses. Dans RSK, les positions sont des tuples (x, y, angle).

tuples.py
# Créer un tuple
position = (0.46, 0.0, 3.14)

# Accéder aux éléments (index commence à 0)
x     = position[0]    # 0.46
y     = position[1]    # 0.0
angle = position[2]    # 3.14

# Décomposer (unpacking)
x, y, angle = position

# Tuple à 2 éléments (balle)
balle = (0.30, -0.15)
bx = balle[0]    # x de la balle
by = balle[1]    # y de la balle

# client.ball est un tuple (x, y) ou None
# client.green1.position est un tuple (x, y)
# client.green1.pose     est un tuple (x, y, angle)

Les listes

Une liste est une séquence modifiable entre crochets :

listes.py
# Créer
robots = ["green1", "green2"]
scores = [0, 0]

# Modifier
scores[0] = 3         # scores = [3, 0]
scores.append(5)      # scores = [3, 0, 5]

# Longueur
print(len(robots))    # 2

# Parcourir
for robot in robots:
    print(robot)

7. Imports

Python est livré avec des modules qu'on peut importer pour avoir plus de fonctionnalités. Pour RSK, tu en as trois principaux.

imports.py
# ── Importer tout un module ──────────────────────────────
import math
import rsk

angle = math.radians(180)    # préfixe obligatoire

# ── Importer une fonction précise ────────────────────────
from math import radians, atan, sqrt, pi

angle = radians(180)         # pas besoin de math. devant

# ── Les trois modules utilisés dans RSK ──────────────────
import rsk              # la bibliothèque Robot Soccer Kit
import math             # fonctions mathématiques (atan, sqrt, pi...)
from math import radians# raccourci pratique pour les angles

# ── Exemple complet ──────────────────────────────────────
import rsk
import math
from math import radians

with rsk.Client(host='localhost', key='') as client:
    while True:
        if client.ball is not None:
            xball = client.ball[0]
            yball = client.ball[1]
            xg1   = client.green1.position[0]
            yg1   = client.green1.position[1]
            # math.atan calcule l'angle vers la balle
            angle = math.atan2(yball - yg1, xball - xg1)

Préfère math.atan2(dy, dx) à math.atan(dy/dx) pour éviter la division par zéro quand le robot est exactement sur la même abscisse que la balle !
