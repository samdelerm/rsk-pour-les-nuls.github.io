C'est quoi RSK ?

Robot Soccer Kit (RSK) est un système de robots miniatures qui jouent au foot sur un terrain vert. Tu les programmes en Python depuis ton ordinateur, ils reçoivent tes ordres en temps réel.

📡
Connexion WiFi ou Simulateur
Tu connectes ton programme Python à un serveur RSK (réel ou simulé). Les robots reçoivent tes commandes en temps réel via WiFi. Pour s'entraîner : host='localhost'.
🤖
2 équipes, 2 robots chacune
Équipe verte : client.green1 et client.green2. Équipe bleue : client.blue1 et client.blue2. Tu contrôles une équipe, le serveur gère l'autre.
📐
Un terrain avec des coordonnées
Le terrain fait 1,84 m × 1,23 m. Le centre est à (0, 0). L'axe X va de −0,92 à +0,92, l'axe Y de −0,615 à +0,615. Tout se mesure en mètres.
🐍
Python + la bibliothèque rsk
Tu écris ton code Python, tu importes rsk, et c'est parti. Pas de matériel complexe à configurer, pas d'électronique — juste du code.

Démarrage rapide

Pour tester sur le simulateur sans robot physique.

premier_programme.py 
import rsk
from math import radians

# Connexion au simulateur (lance d'abord : python -m rsk.simulator)
with rsk.Client(host='localhost', key='') as client:
    while True:
        # Récupérer la position de la balle
        if client.ball is not None:
            bx, by = client.ball[0], client.ball[1]

            # Envoyer le robot 1 vert derrière la balle pour tirer
            if client.green1.goto((bx + 0.08, by, radians(180)), wait=False):
                client.green1.kick()

Lance d'abord le simulateur avec la commande python -m rsk.simulator dans un terminal, puis exécute ton programme dans un autre terminal. Tu verras les robots bouger en direct !
